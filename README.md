# 🔰 MATRIX AI CONSOLE - PYTHON EDITION

**Offline AI Model Manager - Retro 80s Cyberpunk Edition**

## Quick Start

```bash
# Run the Matrix AI Console
python3 /home/capgalius/skynet/ia/matrix_ai.py
```

## Features

- 📦 **List Models** - Browse available Ollama models
- ⚙️ **Profiles** - Manage model profiles and configurations  
- 🎮 **Run Models** - Interactive model selection and execution
- 🔍 **System Status** - Monitor Ollama service and available models
- 🚀 **GitHub Copilot Integration** - Launch Copilot CLI

## Requirements

- Python 3.7+
- [Ollama](https://ollama.ai/) - For model management
- Curl - For API communication

## Configuration

Edit `/home/capgalius/skynet/ia/config` to:
- Define available models
- Set up model profiles
- Configure descriptions
